# ═══════════════════════════════════════════════════════════════════════════════
# 
# Installation order:
#   1. pip install -r scripts/windows/requirements-windows.txt
#   2. pip install -r requirements.txt
#
# Hardware acceleration:
#   - DirectML (GPU/NPU): Included via onnxruntime-directml
#   - CUDA (NVIDIA): Install onnxruntime-gpu instead if you have NVIDIA GPU
#   - CPU-only: Use onnxruntime instead of onnxruntime-directml
#
# macOS → Windows package mapping:
#   - PyAudio → pyaudiowpatch (WASAPI support)
#   - py-applescript → pywin32 (PowerShell/COM)
#   - pyobjc-* → pythonnet (Objective-C → C#/.NET)
#   - CoreML → ONNX Runtime DirectML
#   - Quartz → pywin32 (Win32 API)
#   - AppKit → pywinauto (UI automation)
#
# Known issues:
#   - pywin32 requires "python Scripts/pywin32_postinstall.py -install" after pip install
#     (The install_windows.ps1 script handles this automatically)
#   - onnxruntime-directml requires Windows 10 version 1809 or later
#   - Some packages may require Visual Studio Build Tools for native compilation
#
# Visual Studio Build Tools:
#   Download: https://visualstudio.microsoft.com/downloads/
#   Required components: "Desktop development with C++"
#
